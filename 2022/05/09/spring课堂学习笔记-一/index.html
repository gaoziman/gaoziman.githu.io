Spring

Spring 概述

百度百科解释

用自己的话总结

  • 1.Spring是一个使用java语言开发的轻量级的开源框架

  • 2.Spring可以解决企业应用开发的复杂性

  • 3.Spring不仅可以用于j2ee网站开发，也可以应用于小程序开发和桌面应用程序开发。

  • 4.Spring提供了两个核心功能技术：IOC、AOP

    loC(Inverse of Control) :控制反转
    把创建对象的工作交由Spring完成，Spring在创建对象的时候同时可以完成对象属性赋。

AOP (Aspect Oriented Programming):面向切面
面向切面编程，可以在不改变原有业务逻辑的情况下实现对业务的增强。

  • 5.Spring框架主要由七部分组成，分别是 Spring Core、 Spring AOP、 Spring ORM、 Spring DAO、Spring Context、 Spring Web和 Spring Web MVC。

以上只是简单介绍了spring 框架，这个软件的含义。

哪spring由哪些部分组成，我们如何去学习呢？

现在我们去spring官方网站看看,并把spring框架，这个软件下载下来。
