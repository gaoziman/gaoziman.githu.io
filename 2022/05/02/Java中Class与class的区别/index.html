Java中class与Class的区别

一.class与Class区别

class是Java中的关键字，如public class Xxx 或者 class Xxx ，在声明Java类时使用。 而Class是一个类。 我们通常认为类是对象的抽象和集合，Class就相当于是对类的抽象和集合。 也可以认为对象是类的实例，类是Class的实例。

二.Class介绍

Class是一个类。如下图，它在java.lang包中。 它的构造函数是private属性，所以我们不能直接new一个Class对象出来，如下图注释段所说： “私有构造函数。只有Java虚拟机创建类对象。不使用此构造函数，并阻止生成默认构造函数。”
